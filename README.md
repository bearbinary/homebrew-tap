# Bearbinary Tap

## How do I install these formulae?

`brew install bearbinary/tap/<formula>`

Or `brew tap bearbinary/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bearbinary/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
